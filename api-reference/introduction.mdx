<Note>
  Esta documentação descreve os endpoints da API do projeto {Clubsy}, que é uma plataforma de gestão esportiva para escolinhas e clubes.
</Note>

## Bem-vindo à API do {Clubsy}

A API do {Clubsy} foi projetada para ser modular, escalável e segura, permitindo a integração com múltiplos módulos como **Pagamentos**, **Cadastro**, **Comunicação**, **Notificações**, e mais.

## Autenticação

Todos os endpoints da API são protegidos usando **Bearer Tokens**. Certifique-se de incluir o token de autenticação no cabeçalho de cada requisição:

```json
"security": [
  {
    "bearerAuth": []
  }
]
