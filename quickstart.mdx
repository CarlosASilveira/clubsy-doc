## Configurando o Ambiente de Desenvolvimento

Aqui você aprenderá como configurar localmente tanto o repositório do projeto quanto o da documentação, além de como visualizar e implantar suas alterações.

### Editar e Visualizar Alterações

<AccordionGroup>
  <Accordion icon="github" title="Clone os repositórios localmente">
    Durante o processo de configuração, criamos dois repositórios no GitHub:
    - **Repositório da Documentação**: [https://github.com/CarlosASilveira/clubsy-doc](https://github.com/CarlosASilveira/clubsy-doc)
    - **Repositório do Projeto**: [https://github.com/CarlosASilveira/clubsy-project](https://github.com/CarlosASilveira/clubsy-project)

    Para clonar os repositórios localmente, siga estas [instruções](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) no terminal.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Documentação - Visualize suas alterações">
    A visualização ajuda a garantir que suas alterações estejam conforme o esperado. Utilizamos uma interface de linha de comando para renderizar essas alterações localmente.

    1. Instale o [Mintlify CLI](https://www.npmjs.com/package/mintlify) para visualizar as alterações da documentação localmente com este comando:
       ```bash
       npm i -g mintlify
       ```
    2. Execute o seguinte comando na raiz da documentação (onde está o arquivo `docs.json`):
       ```bash
       mintlify dev
       ```

    <Note>
      Se você estiver usando o antigo arquivo de configuração ```mint.json```, atualize o Mintlify CLI:

      ```bash
      npm i -g mintlify@latest
      ```
      E execute o novo comando de upgrade no repositório da documentação:
      ```bash
      mintlify upgrade
      ```
      Agora você estará usando o novo arquivo de configuração ```docs.json```. Você pode excluir o arquivo ```mint.json``` do repositório.
    </Note>
  </Accordion>
</AccordionGroup>
